对网页进行精细化操作

hexo 常用命令

hexo g -d: 这个命令组合实际上是两个命令的简写。hexo g 是 hexo generate 的缩写，它会生成静态文件；而 -d 参数是 –deploy 的缩写，它会在生成静态文件之后立即部署到配置好的服务器上。这个命令的作用是生成网站静态页面并将其部署。

hexo s: 这个命令是 hexo server 的缩写，它会启动一个本地服务器。这通常用于本地预览，可以在你的浏览器中输入 localhost:4000（默认端口为4000）来查看你的网站。

hexo init [folder]: 创建一个新的Hexo项目。如果提供了文件夹名称，Hexo会在指定的文件夹中创建项目。

hexo new [layout] title: 创建一篇新文章。你可以指定布局（layout），比如post、page等，以及文章的标题。

hexo publish [layout] filename: 发布草稿。
